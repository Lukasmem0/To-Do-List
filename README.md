# React Todo List 

This is a simple Todo List application built using ReactJS that utilizes the browser's local storage to store data.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Persist data in local storage
- Mobile responsive design


## Technologies Used

- HTML
- CSS
- ReactJS
- Local Storage
